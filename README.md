# 大将

## 使い方
`大将！`と呼びかけると使い方を教えてくれるよ

## 機能
- [x] 寿司画像の表示
- [x] 雨雲画像の表示
- [x] Todoist連携

- [x] 料理レシピの紹介
- [x] 周辺のランチマップの表示
